## 数据库的基本操作
查看当前DB: `show databases;`  
添加DB: `create database gc;`  
删除DB: `drop database gc;`

### 创建数据表
```
create table account(
  id bigint(20),
  createTime datetime,
  ip varchar(255),
  brief text,
  gender int(11),
)
```
### 删除数据表
```
drop table account;
```
